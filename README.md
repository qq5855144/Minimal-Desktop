# Minimal Desktop 使用技巧

一款运行在浏览器里的 iOS 风格桌面，支持应用管理、文件夹、组件、隐私桌面与多端同步。

---

## 🚀 基础操作

### 添加应用
点击桌面上的「**添加应用**」系统图标，输入应用名称和网址即可创建新图标。支持自动抓取网站 favicon 作为图标。

### 打开应用
**单击**任意应用图标，在新标签页中打开对应网址。

### 移动图标
**长按**图标（约 0.5 秒）进入拖拽模式，拖至目标位置后松手完成移动。拖动到页面边缘会自动切换到相邻页。

### 切换页面
左右滑动桌面，或点击底部的**分页指示点**跳转到指定页。

### 右键菜单（桌面端）
在图标上**右键单击**可快速访问「编辑」「删除」「移入文件夹」等操作，无需进入长按模式。

---

## 📁 文件夹

### 创建文件夹
将一个图标**拖拽到另一个图标上方**并悬停约 0.8 秒，两者会自动合并为文件夹。

### 打开 / 关闭文件夹
单击文件夹图标展开内容；点击文件夹外部区域或按 Esc 关闭。

### 重命名文件夹
打开文件夹后，**长按**文件夹标题（名称区域）即可进入编辑状态，输入新名称后回车保存。

### 从文件夹移出应用
打开文件夹后，**长按**内部应用图标，拖出到文件夹外的桌面区域即可。

### 文件夹容量
单个文件夹最多容纳 **9 个**应用。超出上限时无法再拖入。

## 🔒 隐私桌面

### 进入隐私桌面
点击底部分页指示区左侧的**🔒 锁形按钮**，首次使用需设置 PIN 码。

### 设置 / 修改 PIN
在「设置 → 隐私桌面」中设置或修改 6 位 PIN 码。PIN 通过 AES-256-GCM 加密存储，无法找回，请牢记。

### 将应用移入隐私桌面
长按图标进入拖拽模式，将图标拖拽到底部**锁形按钮区域**，解锁后图标会进入隐私桌面。

> **注意**：隐私桌面未解锁时，拖入操作会自动中止并跳转到解锁页面，图标保留在原位。

### 将应用移回普通桌面
解锁隐私桌面后，长按隐私页内的图标并拖拽到普通桌面页即可。

### 错误次数锁定
连续输入错误 PIN 5 次后，隐私桌面将锁定 **5 分钟**，请耐心等待后再试。

## 🧩 桌面组件

桌面顶部固定有两个组件行：

| 组件 | 功能 |
|------|------|
| **时钟组件** | 实时显示当前时间与日期 |
| **搜索组件** | 快速搜索，支持切换搜索引擎 |

### 切换搜索引擎
在「**设置 → 搜索引擎**」中选择内置引擎（百度 / Bing / Google 等），或添加自定义引擎（需填入含 `{q}` 占位符的 URL 模板）。

## ⚙️ 设置

点击桌面上的「**设置**」系统图标打开设置面板。

### 外观样式
- **磨砂玻璃（Glassmorphism）**：默认风格，图标背景半透明模糊
- **新拟态（Neumorphism）**：浅色凸起质感

### 壁纸
- 上传本地图片或 GIF 作为壁纸
- 支持视频壁纸（刷新后需重新设置）
- 开启「叠加色彩滤镜」可为壁纸添加极光、日落等色调

### 图标与布局
- **图标大小**：拖动滑块调整（建议 40–56px）
- **每行图标数**：4 列或 5 列
- **每页行数**：1–14 行（增大行数可在一页放更多图标）

### 添加 / 删除页面
在设置中点击「**添加页面**」可新增空白桌面页。

## ☁️ 多端同步

Minimal Desktop 支持通过 **GitHub 私有仓库**同步桌面数据，在不同设备间保持一致。

### 配置步骤
1. 点击「**同步**」系统图标打开同步设置
2. 填入 GitHub **Personal Access Token**（需 `repo` 权限）
3. 指定仓库名、分支和文件路径
4. 点击「**立即上传**」保存当前桌面到云端

### 同步策略

| 模式 | 说明 |
|------|------|
| 手动 | 仅在点击「立即上传 / 下载」时触发 |
| 每日 / 每周 / 每月 | 应用启动时检查，超过间隔则自动同步 |
| 数据变更时自动上传 | 每次修改图标后立即上传 |

> **提示**：下载云端数据会**覆盖**本地桌面，请在下载前确认本地数据已备份或已上传。

## 💡 快捷技巧

- **快速编辑**：右键图标 → 编辑，无需长按进入拖拽模式
- **批量整理**：先建好文件夹，再逐一将图标拖入
- **隐私图标排序**：在隐私桌面内也支持长按拖拽重新排列
- **壁纸 GIF**：上传动态 GIF 可获得动态桌面效果
- **自定义图标**：编辑应用时可粘贴任意图片 URL 替换默认 favicon
